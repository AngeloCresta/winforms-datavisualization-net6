A chart consists of numerous different text elements: 

  • Titles - uses the TitleCollection property. Titles can be placed anywhere on the chart and be used as text or labeling elements.
  • Axis Titles - placed along an axis in the chart area to deLegend Titles - placed in the legend to describe the legend.
  • Axis Labels - displays the axis scale values.
  • Value Labels - displays information about the data.
  • Legend Text - explains series information in the legend.
  • Annotation Text - explanatory notes that can be attached to various chart elements.
